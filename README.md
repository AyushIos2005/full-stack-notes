# 📝 Full Stack Notes

A private full-stack notes application built with the **MERN Stack** for managing my personal notes securely.

This project was created primarily for my own daily use to organize programming notes, project ideas, interview preparation, learning resources, and personal documentation.

🌐 **Live Demo:** https://full-stack-notes-frontend-092f.onrender.com/

---

## 🚀 About the Project

Instead of relying on third-party note-taking services, I wanted a lightweight application that I fully control.

This project allows me to:

* 📝 Create notes
* ✏️ Update notes
* 🗑️ Delete notes
* 🔍 Search notes
* 💾 Store notes in MongoDB
* ⚡ Access my notes from anywhere

Currently, this application is intended **only for my personal use**.

## Getting Started

### Clone

```bash
git clone https://github.com/AyushIos2005/full-stack-notes.git
```

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend/notes-react
npm install
npm run dev
```
